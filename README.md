# SQL + Excel Sales Dashboard

Proyecto de análisis de ventas utilizando SQL y Excel para la visualización y análisis de datos comerciales.

## Tecnologías utilizadas

- MySQL Workbench
- SQL
- Microsoft Excel
- GitHub
- Windows 11

## Análisis realizados

- Clientes con mayor gasto
- Productos más vendidos
- Ventas por categoría
- Productos no vendidos
- Promedio de ventas

## Herramientas de Excel utilizadas

- Tablas dinámicas
- Gráficos
- Dashboard
- Filtros
- Formato de tablas
